# Beauty Constructor – конструктор сайтов-визиток для салонов красоты

Полноценное SaaS-приложение на Next.js для создания, редактирования и публикации сайтов-визиток.

## 🚀 Стек технологий

- **Frontend:** Next.js 14 (App Router), React 18, TypeScript, CSS Modules
- **Backend:** Next.js API Routes, Prisma ORM
- **База данных:** SQLite (локально), PostgreSQL (облачная версия)
- **Аутентификация:** Firebase Auth (Google)
- **Загрузка файлов:** локальное сохранение в public/uploads
- **Уведомления:** react-hot-toast
- **Деплой:** Vercel / ngrok

## ✨ Возможности

- 🎨 **Конструктор услуг** – добавляй, редактируй и удаляй услуги в карточках с ценой, описанием и фото
- 🖼 **Загрузка изображений** – логотип, фото услуг и галерея работ загружаются прямо с устройства
- 💾 **Автосохранение** – форма не теряет данные при случайной перезагрузке страницы
- 🌈 **Стилизация** – 3 пастельных темы (розовая, лавандовая, мятная)
- 📱 **Адаптивность** – конструктор и готовые сайты отлично выглядят на телефоне
- 📋 **Дашборд** – управление сайтами: публикация, копирование ссылки, редактирование, удаление
- 🔔 **Toast-уведомления** – стильные всплывающие сообщения вместо системных alert
- ✨ **Анимации** – плавное появление блоков, пульсация кнопок, анимированные карточки
